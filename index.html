Récuperer les sources

Pour télécharger les sources, rien de plus simple. Disponible sur github, il vous suffit de clone le repository.

git clone git://github.com/Phenix789/Zia.git

Ou d'ajouter le repository en tant que submodule.

git submodule add git://github.com/Phenix789/Zia.git
